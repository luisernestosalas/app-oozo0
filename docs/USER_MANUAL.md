# Manual de Usuario - CRM App Oozo0

## ¿Qué es esta aplicación?

Esta es una aplicación web simple para gestionar clientes y sus notas. Te permite agregar, editar y eliminar información de clientes de forma fácil y rápida.

## Cómo acceder

Abre tu navegador web y ve a:
**https://app-oozo0-m75pryprz-luisernestosalas-2775s-projects.vercel.app**

## Funciones principales

### 1. Ver lista de clientes
- Al abrir la app verás todos tus clientes
- Cada cliente muestra su información básica

### 2. Agregar nuevo cliente
- Busca el botón "Agregar Cliente" o "+"
- Completa los campos requeridos
- Haz clic en "Guardar"

### 3. Editar cliente existente
- Haz clic en el cliente que quieres modificar
- Cambia la información necesaria
- Guarda los cambios

### 4. Eliminar cliente
- Selecciona el cliente
- Busca la opción "Eliminar" (normalmente un ícono de basura)
- Confirma la eliminación

### 5. Gestionar notas
- Dentro de cada cliente puedes:
  - Agregar nuevas notas
  - Editar notas existentes
