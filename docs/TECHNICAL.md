# Documentación Técnica - CRM App

## Información General
- **Aplicación**: CRM con gestión de clientes y notas
- **Stack Tecnológico**: HTML + JavaScript (Frontend)
- **Repositorio**: https://github.com/luisernestosalas/app-oozo0
- **URL de Producción**: https://app-oozo0-m75pryprz-luisernestosalas-2775s-projects.vercel.app
- **Plataforma de Deploy**: Vercel

## Arquitectura
- **Tipo**: Single Page Application (SPA)
- **Frontend**: HTML5 + JavaScript vanilla
- **Almacenamiento**: Local Storage (navegador)
- **Hosting**: Vercel

## Funcionalidades Principales
1. **Gestión de Clientes**
   - Crear nuevos clientes
   - Visualizar lista de clientes
   - Editar información de clientes
   - Eliminar clientes

2. **Sistema de Notas**
   - Agregar notas a clientes
   - Visualizar historial de notas
   - Editar notas existentes
   - Eliminar notas
