# CRM con Clientes y Notas

Un sistema de gestión de relaciones con clientes (CRM) simple y eficiente desarrollado con tecnologías web básicas.

## 🚀 Características

- **Gestión de Clientes**: Agregar, editar y eliminar información de clientes
- **Sistema de Notas**: Crear y gestionar notas asociadas a cada cliente
- **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- **Almacenamiento Local**: Los datos se guardan en el navegador
- **Responsive**: Compatible con dispositivos móviles y desktop

## 🛠️ Stack Tecnológico

- **HTML5**: Estructura y semántica
- **JavaScript ES6+**: Lógica de la aplicación
- **CSS3**: Estilos y diseño responsivo
- **LocalStorage**: Persistencia de datos
